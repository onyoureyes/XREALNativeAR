# 변경 요청 (REQUESTS)

> 다른 모듈 파일 변경이 필요할 때 여기에 요청을 작성한다.
> 지휘본부(사용자)가 DECISIONS.md에서 승인할 때까지 대기.
> 차단되면 Mock/Fake로 우회하며 진행한다.

## 형식

```
### REQ-번호 | 상태
- 요청자: Track X (모듈명)
- 대상: :대상모듈 (Track Y 영역)
- 내용: 구체적 변경 내용
- 이유: 왜 필요한지
- 긴급도: 차단됨(이거 없으면 진행 불가) | 높음 | 낮음(나중에 해도 됨)
```

## 상태값
- **OPEN**: 요청 작성됨, 지휘본부 검토 대기
- **APPROVED**: 승인됨, 대상 Track이 실행 가능
- **DENIED**: 거부됨, 사유는 DECISIONS.md 참조
- **DONE**: 변경 완료, CHANGELOG.md에 기록됨
